ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА:

1. Установи библиотеки (если ещё не установлены):
   pip install python-telegram-bot pandas openpyxl

2. Убедись, что promo.pdf лежит в той же папке.

3. Запусти бота командой:
   python bot.py

4. Готово! Теперь можно перейти к боту через ссылку или QR-код.

Если что-то не работает — проверь токен и интернет.
